

# My Flutter Portfolio Website

A modern, interactive portfolio website built with **Flutter Web**, featuring:

- Blurred background 
- Dark mode toggle
- Full-page scrollable sections
- Tabbed navigation for organized content
- Responsive design

---

## üåê Features

- **Full-screen background**: The background image stays fixed and blurred, giving a sleek modern look.
- **Glassmorphism**: Sections have semi-transparent containers with blur and borders.
- **Dark/Light Mode**: Toggle between themes using the ThemeProvider.
- **Vertical scrolling & Snap sections**: Each portfolio section takes full viewport height, scrolls smoothly.
- **Tab Navigation**: For Home sections with multiple content pages (Overview, Skills, Projects, etc.).
- **Responsive Design**: Works on mobile, tablet, and desktop web.
